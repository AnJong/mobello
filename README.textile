h1. Mobello, HTML5 framwork for building mobile webapp

Mobello is HTML5 framwork for building mobile webapp. 

h1. Features

h3. UI Components

!{width:250px}http://mobello.github.com/img/feature-02.png!

Mobello includes a complete set of user interface components for building rich mobile web apps.

* 20+ beautifully designed components
* Support a wide range of device form-factor
* Optimized for touch UI
* Easily extends base components and creates new custom component

h3. Mobello Runtime 

!{width:250px}http://mobello.github.com/img/feature-01.png!

Runtime manages the lifecycle of installed apps and provides abstractions for interaction and animation, advanced effects.

* Device indepandant event system (touch events)
* Provides advanced amimation
* Run-time system for multi-app running
* Provides controllers for general ux patterns.

h3. Mobello Studio (coming soon)

!{width:250px}http://mobello.github.com/img/feature-04.jpg!

* Design standard web with HTML5 and CSS3
* Visual Design Editor
* Real-time synchronized editing
* Javascript Debugging
* Device Emulator
* Layout Editing
* Eclipse based
